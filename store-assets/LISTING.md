# Chrome ウェブストア 提出キット（v0.7.1〜）

アップロードする zip: `npm run build && npm run package` → `zen-study-stats-<version>.zip`
スクリーンショット: このフォルダの `01`〜`05`（1280x800 PNG・匿名化デモデータ）

## ストア掲載情報（コピペ用）

**名前**: 学習統計 for ZEN Study

**概要（132字以内）**:
自分の学習データを見やすく可視化する非公式・表示専用ツール。学習時間の実測、完了予測（モンテカルロ）、締切警告、教科別の残り、ダークモード。read-only／GETのみで学習記録は一切変更しません。

**詳細説明**:
ZEN Study の自分の学習データを、より見やすく・より深く可視化する非公式の拡張機能です。
/setting の学習数パネルを高機能カードに置き換え、コース/章画面に残り作業量を表示します。

【主な機能】
・学習時間の実測（アクティブタイム方式）と日別/時間帯グラフ
・年度レポートの完了予測（時間ベースのモンテカルロ・間に合う確率）
・月次レポート締切の警告と「このペースだと間に合わない教科」判定
・教科別の残り（教材数と時間換算）・章別内訳
・学習カレンダー（GitHub風）・トレンド・曜日/時間帯の傾向
・今日/今週の目標・連続学習・達成バッジ
・サイト全体のダークモード
・データのバックアップ/復元（JSON/CSV・ローカルのみ）

【安全性・プライバシー】
・完全 read-only: 取得は GET のみ。学習記録の変更・動画進捗の操作は一切しません
・すべての処理と保存はブラウザ内で完結。外部サーバーへの送信・解析・広告は一切ありません
・ソースコード公開: https://github.com/mug1ch4/zen-study-stats

【重要なお知らせ】
・本拡張機能は非公式の個人プロジェクトで、KADOKAWA・ドワンゴ・ZEN Study 運営とは関係ありません
・ZEN Study 運営はサードパーティ製ツールの個別の検証・許諾を行っていません。利用規約の範囲内で自己責任でご利用ください
・本家サイトの不具合を運営に問い合わせる際は、本拡張機能を無効化して再現確認をしてからにしてください
・ZEN Study へのログインが必要です（自分のアカウントの自分のデータのみ表示します）

**カテゴリ**: 仕事効率化（ツール）
**言語**: 日本語

## ダッシュボード入力（プライバシー関連）

- **単一用途の説明**: ZEN Study 上の自分の学習データを読み取り専用で集計・可視化する
- **権限の理由**:
  - `storage`: 学習履歴の集計値（日別学習数・学習時間など）を端末内に保存するため
  - ホスト権限 `www.nnn.ed.nico` / `api.nnn.ed.nico`: ログイン中ユーザー自身の学習データを GET で取得し、ページ内にカードを表示するため
- **リモートコード**: 使用しない
- **データ使用**: 「ユーザーデータを収集または使用しない」…ではなく正確には
  「ウェブサイトコンテンツ（学習データ）を扱うが、**外部送信・第三者提供・売却は一切なし**」として、
  データ収集の開示では該当カテゴリ（個人情報等）は**すべて「収集しない」**にチェック
  （拡張内ローカル処理のみは「収集」に該当しない）。認証情報も収集しない（既存セッションを利用するだけ）
- **プライバシーポリシーURL**: https://mug1ch4.github.io/zen-study-stats/privacy.html

## 提出手順

1. https://chromewebstore.google.com/devconsole → Googleアカウントでデベロッパー登録（初回 $5）
2. 「新しいアイテム」→ `zen-study-stats-<version>.zip` をアップロード
3. ストア掲載情報タブ: 上記の名前/概要/詳細/カテゴリ/言語・スクショ5枚・アイコン128px（zipに同梱済みだが掲載用にも `icons/icon128.png` を使用）
4. プライバシータブ: 上記の単一用途・権限理由・データ開示・ポリシーURL
5. 配布タブ: 公開（一般公開）・地域は全て（または日本のみでも可）
6. 審査へ提出（数日〜1週間程度。ホスト権限があるため人手審査になりやすい）
