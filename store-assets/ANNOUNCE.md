ZEN Study の自分の学習データを可視化する Chrome 拡張を作ったので共有します 📊

**学習統計 for ZEN Study**（非公式・無料・オープンソース）
▶ Chrome ウェブストア: https://chromewebstore.google.com/detail/bkmampnickklonnmenhclffblfeojndd
▶ 触って試せるデモ（サンプルデータ）: https://mug1ch4.github.io/zen-study-stats/

できること:
・学習時間の実測（1日何時間勉強したか・時間帯の傾向）
・年度レポートの完了予測（「このペースなら◯月◯日に完了・間に合う確率◯%」）
・月次締切の警告と「このペースだと間に合わない教科」の判定
・教科別の残り（教材数と時間換算）・学習カレンダー・ダークモード など

安全性について（大事なので明記します）:
・**完全 read-only**。取得は GET のみで、学習記録の変更・動画進捗の操作は一切しません
・データはすべて自分のブラウザ内に保存。**外部送信・解析・広告ゼロ**
・ソース全公開: https://github.com/mug1ch4/zen-study-stats — 中身を確認してから入れられます

注意事項:
・**非公式の個人プロジェクト**です。ZEN Study 運営に問い合わせたところ「サードパーティ製ツールの個別の検証・許諾は行っていない。利用規約の範囲内で自己責任で」との回答でした
・拡張を入れた状態の不具合は運営サポート対象外になるため、**本家の不具合を問い合わせる時は拡張をオフにして再現確認してから**にしてください
・個人開発なのでサポートはベストエフォートです。不具合報告・要望は GitHub の Issue へお願いします

自分の勉強管理のために作ったものですが、よければどうぞ。
